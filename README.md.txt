# Yayıncı Avı Projesi

Bu projede, yayın platformları kullanıldığında ağ üzerinden geçen trafiği Wireshark ile analiz ettim.  
Özellikle YouTube canlı yayını izlenirken oluşan HTTPS trafiği incelendi.

## Amaç
- Yayın trafiğinde kullanılan IP ve port bilgilerini tespit etmek
- Yakalanan verileri `.pcapng` formatında kaydetmek
- Analiz sırasında ekran görüntüsü almak

## Araçlar
- Wireshark
- YouTube (Canlı yayın testleri için)
- GitHub (rapor ve çıktıların sunumu)

## İçerik
- `trafik.pcapng`: Canlı yayın izlenirken yakalanan ağ trafiği
- `ekran-goruntusu.png`: Wireshark ekranından alınan örnek görüntü
